## 权限和变量

- 终端和脚本对环境变量的处理不太一样：
  - 终端中`sudo`提权后很多环境变量仍不是root用户的变量
  - 使用`sudo`运行脚本，脚本中使用的环境变量就是root环境下的了

  > 所以对于对环境变量有要求的脚本，应该添加User Check功能

## 重定向

### 输出重定向

1. 标准输出

    ```shell
    命令 > 文件     # 清空模式
    命令 >> 文件    # 追加模式
    ```

2. 错误输出

    ```shell
    命令 2> 文件     # 清空模式
    命令 2>> 文件    # 追加模式
    ```

3. 标准输出和错误输出

    ```shell
    命令 &> 文件     # 清空模式
    命令 &>> 文件    # 追加模式
    ```
